📖 Introducción

📋 Requisitos del Sistema

  • Navegador web moderno (Chrome, Firefox, Safari, Edge)

🚀 Instalación y Configuración

Opción 1: Instalación Simple (Recomendado)

Si solo deseas usar el theme compilado sin modificar el código:

  1. Extrae todos los archivos en tu servidor web
  2. Abre el archivo index.html en tu navegador
  3. ¡Listo! El theme está funcionando
✓ Nota: No necesitas instalar Node.js o cualquier otra dependencia. Los archivos ya están compilados y listos para usar.

Opción 2: Desarrollo Avanzado (Con Node.js)

Si deseas modificar el código y compilar cambios:

  1. Asegúrate de tener Node.js v16+ instalado
  2. Instala las dependencias: 
    npm install
  3. Inicia el servidor de desarrollo: 
    npm run dev
  4. El tema estará disponible en http://localhost:5173

Compilación para Producción

npm run build

Los archivos optimizados se generarán en la carpeta dist/

Requisitos para Desarrollo

  • Node.js - v16 o superior
  • npm - Incluido con Node.js
  • Editor de código - VS Code, Sublime Text, etc.

🎨 Personalización

Cambiar Colores Principales

Los colores principales se encuentran en el archivo src/style.css. Busca las variables CSS:

:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --accent-color: #f093fb;
  --text-dark: #333333;
  --text-light: #ffffff;
  --bg-light: #f8f9fa;
}

Modifica estos valores para cambiar la paleta de colores del tema.

Cambiar Fuentes

Para cambiar las fuentes, edita el archivo index.html en la sección <head>:

<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT:wght@400;600;700&display=swap" rel="stylesheet">

Luego actualiza el font-family en src/style.css.

Cambiar Logo y Branding

  • Reemplaza el texto "Estudio" en la barra de navegación por tu nombre de marca
  • Cambia el favicon en public/favicon.svg
  • Actualiza el título de la página en <title>
  • Modifica las imágenes de fondo en cada sección

Cambiar Secciones y Contenido

El contenido está organizado en secciones dentro del index.html:

// Secciones principales:
- Header/Hero
- Experiencia
- Trabajos
- Testimoniales
- Contacto

Puedes editar el texto, imágenes y estructura de cada sección directamente en el HTML.

Personalización Avanzada

Si necesitas realizar cambios más avanzados:

  1. Edita los archivos en la carpeta src/
  2. Ejecuta npm run build para compilar
  3. Los archivos compilados aparecerán en la carpeta dist/

❓ Preguntas Frecuentes (FAQ)

¿Necesito conocimientos de programación?

No necesariamente. Puedes cambiar texto, imágenes y colores solo editando el HTML. Sin embargo, conocimientos básicos de HTML/CSS te permitirán personalizaciones más avanzadas.

¿Cómo cambio las imágenes de fondo?

En el archivo index.html, busca los elementos con bg-[url(...)] y reemplaza la URL con tu imagen:

<div class="bg-[url('tu-imagen.jpg')]"></div>

¿Es compatible con navegadores antiguos?

El theme es compatible con navegadores modernos (Chrome, Firefox, Safari, Edge de los últimos 2 años). Para compatibilidad con navegadores más antiguos, se recomienda usar polyfills.

¿Puedo usar este theme comercialmente?

Sí, está incluido en tu licencia de compra. Puedes usarlo en proyectos personales y comerciales.

¿Cómo añado un formulario de contacto funcional?

El theme incluye un formulario HTML básico. Para hacerlo funcional, necesitas:

  1. Usar un servicio como FormSubmit, Netlify Forms, o Firebase
  2. O configurar un backend propio
  3. Ver la sección Soporte para más detalles

¿Cómo optimizo el SEO?

  • Usa meta tags descriptivos en <head>
  • Incluye palabras clave relevantes en los títulos
  • Usa encabezados HTML correctamente (h1, h2, h3)
  • Optimiza las imágenes (comprime y usa alt text)
  • Asegúrate de que el sitio sea rápido

¿Cómo agrego más secciones?

Copia una sección existente en el index.html, modifica el contenido y añade tu propio estilo si es necesario.

¿Por qué algunas animaciones no se ven en móvil?

Es normal. AOS (Animate On Scroll) se deshabilita automáticamente en dispositivos móviles para mejor rendimiento. Puedes cambiar esto en src/main.js.
