# Resonance - Architecture Theme 🏛️

Un tema moderno y responsivo para sitios web de estudios de arquitectura, despachos creativos y profesionales del diseño, desarrollado con tecnologías web actuales.

---

## 🚀 Características

- **Diseño responsivo** compatible con móviles, tablets y escritorio.
- **Tailwind CSS** para una interfaz moderna y personalizable.
- **Animaciones suaves** con AOS (Animate On Scroll).
- **Menú mobile** con navegación hamburguesa optimizada.
- **Vite** como entorno rápido de desarrollo y build.
- **Código limpio y estructurado**.

---

## 📋 Requisitos Previos

- Node.js **v16 o superior**
- npm o yarn
